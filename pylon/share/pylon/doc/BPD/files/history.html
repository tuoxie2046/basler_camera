History#

This topic provides information about the current and previous versions of the Basler Product Documentation.

Version 46 (February 2021)#

  • Renamed the "Embedded Vision Kits" section "Embedded Vision".
  • Moved the Basler dart BCON for MIPI camera module documentation to the "Embedded Vision" section.
  • Revised and restructured the BCON for MIPI camera module documentation.
  • Updated pin names and functions in all dart BCON for MIPI camera module topics.
  • Added information about external triggering and the NVIDIA Jetson Xavier NX developer board in the following Add-On Camera Kits topics:
    • daA2500-60mci-NVJET-NVDK-AddOn
    • daA4200-30mci-NVJET-NVDK-AddOn
    • daA4200-30mci-JNANO-NVDK
  • Updated design and wording of the Frame Grabbers and VisualApplets documentation in line with Basler's documentation style.
  • Revised the Frame Grabbers topic.
  • Added the "Models" section and the Frame Grabber Models topic to the frame grabber documentation.
  • Added KC compliance to the specification tables of the boA6500-36cm and boA8100-16cm camera models.
  • Updated mechanical drawings of the Basler boost CXP-12 camera series.
  • Corrected the values for the maximum allowed lens intrusion for all Basler boost CXP-12 camera models and updated figures.
  • Removed deprecated C.NET code samples in all camera feature topics.
  • Replaced "C++ (Generic)" code samples with the newer samples previously found under "C++ (Generic, v5.2+)".
  • Corrected the filter size of the Basler Lens C11-1220-12M.
  • Slightly revised the Event Notification topic.
  • Slightly revised the Binning topic. Updated the binning selector availability and parameter values for the following cameras: ace 2 mono, boost mono, and daA1920-160um
  • Added cable lengths to cable drawings in all SLP cable topics.
  • Added information about downloading an extra driver for the 10GigE Interface Card, 1 Port from www.marvell.com.
  • Removed operating systems that are no longer supported from the Software Installation (Windows) and the Software Installation (Linux) topics.
  • Updated download links for USB 3.0 host controller drivers in the Hardware Installation (USB 3.0 Cameras) topic.
